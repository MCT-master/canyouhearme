Zoom

Official website - Support link

Zoom is a widely used confrencing application for casual and commerical use. It is well supported and compatible with most platforms.
  • Column
  • Table

Basic information


Developer: Zoom US

Protocol: Audio-video

OS Supported: Cross-platform

Pricing: Free (45 meeting host limit), Pro (available for students) or subscription

Requires sign-up?: Yes for hosting, no for joining

Website: https://zoom.us/

Detailed information


Self-hosted?: Server-based

Recommended net-speed: Flexible

Required net-speed: None

Max quality:

Latency (estimate): ~150ms

Last update: Regular updates

Open source?: No

Repo: None

Ease of use

Well developed by a large corp, quite easy to use, install and connect. A Pro account (available for students) can host meetings at infinite time, 45 minutes for free users.



info

There are some important settings that can be optimized for higher quality audio. In settings, 'HD video' in the Video tab and 'Enable original audio', 'High fidelity music mode', 'Stereo audio' in 'Audio'.
